Équations du premier degré

Une équation du premier degré se présente sous la forme :

ax + b = 0

Où a et b sont des nombres réels et x est l'inconnue.

Les propriétés des équations du premier degré sont les suivantes :

  • Toute équation du premier degré a une solution unique.
  • Si a est différent de 0, alors l'inconnue x est égale à -b/a.

Les règles pour résoudre une équation du premier degré sont les suivantes :

  • Isoler x en faisant les opérations inverses des termes qui ne contiennent pas x.
  • Vérifier la solution en remplaçant x par la valeur trouvée dans l'équation de départ.

Voici un exemple :

2x + 5 = 11

Pour isoler x, on soustrait 5 des deux côtés :

2x = 6

Ensuite, on divise par 2 des deux côtés :

x = 3

Pour vérifier la solution, on remplace x par 3 dans l'équation de départ :

2(3) + 5 = 11

Donc x = 3 est bien une solution de l'équation 2x + 5 = 11.


Équations du deuxième degré

Une équation du deuxième degré se présente sous la forme :

ax² + bx + c = 0

Où a, b et c sont des nombres réels et x est l'inconnue.

Les propriétés des équations du deuxième degré sont les suivantes :

  • Une équation du deuxième degré peut avoir deux solutions réelles, une solution double ou deux solutions complexes conjuguées.
  • Si delta = b² - 4ac est positif, l'équation a deux solutions réelles distinctes : x1 = (-b + √(delta))/2a et x2 = (-b - √(delta))/2a.
  • Si delta = 0, l'équation a une solution double : x = -b/2a.
  • Si delta est négatif, l'équation a deux solutions complexes conjuguées : x1 = (-b + i*√(-delta))/2a et x2 = (-b - i*√(-delta))/2a.

Les règles pour résoudre une équation du deuxième degré sont les suivantes :

  1. Calculer delta = b² - 4ac.
  2. En fonction du signe de delta, calculer les solutions x1 et x2 en utilisant les formules appropriées.

Voici un exemple :

2x² + 5x - 3 = 0

Calculons d'abord delta :

delta = 5² - 4*2*(-3) = 49

Comme delta est positif, l'équation a deux solutions réelles distinctes :

x1 = (-5 + √(49))/4 = 1/2

x2 = (-5 - √(49))/4 = -3

Donc les solutions de l'équation 2x² + 5x - 3 = 0 sont x1 = 1/2 et x2 = -3.

on a aussi : x' + x" = -b/a ; x' * x" = c/a


Table de signe d'une équation du deuxième degré

Pour représenter la table de signe d'une équation du deuxième degré de la forme ax² + bx + c = 0, voici les étapes à suivre :

  1. Calculer les racines de l'équation.
  2. Déterminer les intervalles sur lesquels la fonction associée à l'équation est positive ou négative.
  3. Remplir la table de signe en indiquant les valeurs de x et le signe de la fonction associée à l'équation.

Voici un exemple :

Représenter la table de signe de l'équation x² - 3x + 2 = 0

Calculons d'abord les racines de l'équation :

x1 = 1 et x2 = 2

Les intervalles sur lesquels la fonction associée à l'équation est positive ou négative sont :

  • Sur l'intervalle ]-∞, 1[, la fonction est positive.
  • Sur l'intervalle ]1, 2[, la fonction est négative.
  • Sur l'intervalle ]2, +∞[, la fonction est positive.

La table de signe correspondante est la suivante :

x -∞ 1 2 +∞
f(x) + 0 - +

Règles des suites arithmétiques et géométriques

Suite arithmétique

Définition : Une suite arithmétique est une suite dans laquelle chaque terme est obtenu en ajoutant un nombre constant à son prédécesseur.

Terme général : un = u1 + (n-1)d

Formule de somme : Sn = n/2 (u1 + un) = n/2 (2u1 + (n-1)d)

Suite géométrique

Définition : Une suite géométrique est une suite dans laquelle chaque terme est obtenu en multipliant le terme précédent par un nombre constant.

Terme général : un = u1 * r^(n-1)

Formule de somme (pour r ≠ 1) : Sn = u1(1 - r^n) / (1 - r)


Les fonctions mathématiques

Une fonction mathématique est une relation qui associe chaque élément d'un ensemble de départ (appelé domaine) à un élément d'un ensemble d'arrivée (appelé co-domaine). Elle peut être représentée sous forme d'équation :

f(x) = y

La variation d'une fonction

La variation d'une fonction représente la manière dont elle évolue sur son domaine. Pour déterminer si une courbe est croissante ou décroissante, on peut calculer sa dérivée première :

f'(x) = lim(h→0) [f(x+h) - f(x)] / h

Si f'(x) > 0, la fonction est croissante. Si f'(x) < 0, la fonction est décroissante. Si f'(x) = 0, la fonction atteint un extremum local.

La parité d'une fonction

Une fonction est dite paire si pour tout x de son domaine, f(-x) = f(x). Elle est dite impaire si pour tout x de son domaine, f(-x) = -f(x). Pour déterminer si une fonction est paire ou impaire, on peut utiliser les propriétés suivantes :

  • Une fonction est paire si et seulement si sa courbe est symétrique par rapport à l'axe des ordonnées.
  • Une fonction est impaire si et seulement si sa courbe est symétrique par rapport à l'origine.

Le vecteur directeur et le vecteur normal

Un vecteur directeur est un vecteur qui indique la direction d'une droite ou d'un plan. Pour une droite passant par les points (x1,y1) et (x2,y2), le vecteur directeur est :

(x2 - x1, y2 - y1)

Un vecteur normal est un vecteur qui est perpendiculaire à une droite ou un plan donné. Pour une droite passant par les points (x1,y1) et (x2,y2), un vecteur normal est :

(y1 - y2, x2 - x1)
